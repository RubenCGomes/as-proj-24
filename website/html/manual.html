Manual OneBox

This how you reserve a locker in Onebox

1st Step

Access the lockers page and choose a locker of your choice that matches the size of the package you ordered.

2nd Step

Using the calendar on the right side of the locker, you will be able to define the time interval in which you will need the locker for the order to arrive. Select the start date and end date and apply it

3rd Step

After rentign the locker you are guided to a payment page, there you write all your personal data needed and the payment method. You also can add promo codes or the card code to get discounts on the locker
